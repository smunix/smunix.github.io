In calculus, the quotient rule is a method of finding the derivative of a function that is the quotient of two other functions for which derivatives exist.[1][2][3]

If the function one wishes to differentiate, f ( x ) {\displaystyle f(x)} , can be written as

f ( x ) = g ( x ) h ( x ) {\displaystyle f(x)={\frac {g(x)}{h(x)}}}

and h ( x ) ≠ 0 {\displaystyle h(x)\not =0} , then the rule states that the derivative of g ( x ) / h ( x ) {\displaystyle g(x)/h(x)} is

f ′ ( x ) = g ′ ( x ) h ( x ) − h ′ ( x ) g ( x ) [ h ( x ) ] 2 . {\displaystyle f'(x)={\frac {g'(x)h(x)-h'(x)g(x)}{[h(x)]^{2}}}.}

Many people remember the Quotient Rule by the rhyme "Low D-high, High D-low, cross the line and square below." It is important to remember the 'D' describes the succeeding portion of the original fraction.

Proof using implicit differentiation[edit]

Let f ( x ) = g ( x ) h ( x ) {\displaystyle f(x)={\frac {g(x)}{h(x)}}}
Then g ( x ) = f ( x ) h ( x )   {\displaystyle g(x)=f(x)h(x){\mbox{ }}\,}
Using product rule, g ′ ( x ) = f ′ ( x ) h ( x ) + f ( x ) h ′ ( x )   {\displaystyle g'(x)=f'(x)h(x)+f(x)h'(x){\mbox{ }}\,}
f ′ ( x ) = g ′ ( x ) − f ( x ) h ′ ( x ) h ( x ) = g ′ ( x ) ⋅ h ( x ) h ( x ) − g ( x ) h ( x ) ⋅ h ′ ( x ) h ( x ) {\displaystyle f'(x)={\frac {g'(x)-f(x)h'(x)}{h(x)}}={\frac {g'(x)\cdot {\frac {h(x)}{h(x)}}-{\frac {g(x)}{h(x)}}\cdot h'(x)}{h(x)}}}
f ′ ( x ) = g ′ ( x ) h ( x ) − g ( x ) h ′ ( x ) ( h ( x ) ) 2 {\displaystyle f'(x)={\frac {g'(x)h(x)-g(x)h'(x)}{\left(h(x)\right)^{2}}}}

Proof using chain rule[edit]

f ( x ) = g ( x ) h ( x ) {\displaystyle f(x)={\frac {g(x)}{h(x)}}}

We rewrite the fraction using a negative exponent.

f ( x ) = g ( x ) ⋅ h ( x ) − 1 {\displaystyle f(x)=g(x)\cdot h(x)^{-1}}

Take the derivative of both sides, and apply the product rule to the right side.

f ′ ( x ) = g ′ ( x ) ⋅ h ( x ) − 1 + g ( x ) ⋅ ( h ( x ) − 1 ) ′ {\displaystyle f'(x)=g'(x)\cdot h(x)^{-1}+g(x)\cdot (h(x)^{-1})'}

To evaluate the derivative in the second term, apply the chain rule, where the outer function is x − 1 {\displaystyle x^{-1}} , and the inner function is h ( x ) {\displaystyle h(x)} .

f ′ ( x ) = g ′ ( x ) ⋅ h ( x ) − 1 + g ( x ) ⋅ ( − 1 ) h ( x ) − 2 h ′ ( x ) {\displaystyle f'(x)=g'(x)\cdot h(x)^{-1}+g(x)\cdot (-1)h(x)^{-2}h'(x)}

Rewrite things in fraction form.

f ′ ( x ) = g ′ ( x ) h ( x ) − g ( x ) ⋅ h ′ ( x ) [ h ( x ) ] 2 {\displaystyle f'(x)={\frac {g'(x)}{h(x)}}-{\frac {g(x)\cdot h'(x)}{[h(x)]^{2}}}}
f ′ ( x ) = g ′ ( x ) ⋅ h ( x ) − h ′ ( x ) ⋅ g ( x ) [ h ( x ) ] 2 {\displaystyle f'(x)={\frac {g'(x)\cdot h(x)-h'(x)\cdot g(x)}{[h(x)]^{2}}}}

Higher order formulas[edit]

It is much easier to derive higher order quotient rules using implicit differentiation. For example, two implicit differentiations of f h = g {\displaystyle fh=g} yields f ″ h + 2 f ′ h ′ + f h ″ = g ″ {\displaystyle f''h+2f'h'+fh''=g''} and solving for f ″ {\displaystyle f''} yields

f ″ = g ″ − 2 f ′ h ′ − f h ″ h . {\displaystyle f''={\frac {g''-2f'h'-fh''}{h}}.}
